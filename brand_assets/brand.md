# Norr Studio — Brand Guidelines

## Colors

| Role | Name | Hex |
|------|------|-----|
| Primary background | Norr Black | `#111111` |
| Light background | Norr White | `#FAFAFA` |
| Accent | Nordic Green | `#00E5A0` |
| Accent (light mode) | Nordic Green Dark | `#00A372` |
| Secondary text | Norr Gray | `#888888` |
| Subtle text | Muted | `#444444` |
| Card/surface dark | Surface | `#1A1A1A` |
| Border subtle | Border | `#222222` |

## Typography

**Primary font:** Geist  
**Fallback:** Inter, Arial, sans-serif  
**Available in Next.js via:** `next/font/google` → `Geist`

| Element | Size | Weight | Color |
|---------|------|--------|-------|
| Display headline | 64px | 700 | `#FFFFFF` |
| H1 | 48px | 700 | `#FFFFFF` |
| H2 | 36px | 600 | `#FFFFFF` |
| H3 | 24px | 600 | `#FFFFFF` |
| Body | 16px | 400 | `#FAFAFA` |
| Small / caption | 12px | 400 | `#888888` |
| Eyebrow label | 11px | 500 | `#00E5A0` |

**Letter spacing:** Eyebrow labels use `tracking-widest`. Headings use default or `tracking-tight`.

## Logo

- Logo file: `Web-Icon-512_512x512.png` (white mark + NORR wordmark)
- Always use on `#111111` dark or `#FAFAFA` light backgrounds only
- Minimum clear space: equal to the height of the N on all sides
- Never stretch, rotate or recolor the logo
- Eyebrow text format: `● NORR STUDIO` in `#00E5A0`, small spaced caps

## UI Components

### Buttons
- **Primary:** background `#00E5A0`, text `#111111`, rounded-full, px-6 py-3
- **Secondary:** border `1px solid #333333`, text `#FFFFFF`, transparent background, rounded-full
- **Hover primary:** opacity 90%

### Pills / Tags
- Background: `#00E5A0` at 10% opacity → `rgba(0, 229, 160, 0.1)`
- Text: `#00E5A0`
- Border: `1px solid rgba(0, 229, 160, 0.2)`
- Border radius: `rounded-full`
- Padding: `px-4 py-1.5`
- Font size: `text-sm`

### Cards / Surfaces
- Background: `#1A1A1A`
- Border: `1px solid #222222`
- Border radius: `rounded-xl`

## Spacing

- Section padding: `py-24` (96px)
- Container max width: `max-w-6xl mx-auto px-6`
- Element gaps: `gap-4` (16px) small, `gap-8` (32px) medium, `gap-16` (64px) large

## Brand Voice

- Confident, minimal, direct
- No fluff or filler words
- Short sentences
- Finnish market — professional but approachable
- Never use exclamation marks
